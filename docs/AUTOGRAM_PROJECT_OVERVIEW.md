# SNS 자동화 프로젝트 (Autogram) - AI 전달용 설명서

## 프로젝트 개요

**Instagram 댓글 트리거 기반 자동 DM 발송 시스템**입니다. 비즈니스 계정의 게시물에 특정 키워드가 포함된 댓글이 달리면 자동으로 DM을 발송합니다.

---

## 주요 기능

### 1. 인증 시스템
- 이메일/비밀번호 로그인
- Google OAuth 로그인
- Instagram Business 계정 연동 (필수)
- JWT 기반 토큰 인증 (Access 15분 / Refresh 7일)

### 2. 트리거 관리
- Instagram 게시물 선택 → 트리거 단어 설정 → DM 메시지 작성
- 복수 트리거 단어 지원 (쉼표 구분: "정보,신청,문의")
- 팔로워 필터링 옵션 (팔로워에게만 DM 발송)
- CTA 버튼 추가 (제목 + URL)

### 3. 자동 DM 발송
- Instagram Webhook으로 실시간 댓글 감지
- 중복 발송 방지
- 발송 상태: PENDING → SENT / FAIL / DUPLICATED

### 4. 대시보드 & 통계
- 오늘 발송량, 활성 트리거 수, 배달율
- 시간별 발송 현황 차트
- 상위 트리거 성공률
- 발송 이력 조회 (필터링, 페이지네이션)

---

## 핵심 플로우

```
1. 사용자 로그인 → Instagram 비즈니스 계정 연동
2. 게시물 선택 → 트리거 생성 (키워드 + DM 메시지)
3. Instagram Webhook → 댓글 감지 → 키워드 매칭
4. 조건 충족 시 → 자동 DM 발송 → 이력 저장
5. 대시보드에서 통계 및 이력 확인
```
